### 🛠️ Como usar? / How to use?
1.  **Download:** Descarregue o ficheiro `.zip` do repositório.
2.  **Extract:** Extraia os ficheiros para uma pasta.
3.  **Run:** Arraste o ficheiro `index.html` para o seu navegador de preferência.
